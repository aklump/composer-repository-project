# Composer Repository

## Install

* `{{ composer.create_project|raw }}`
* `bin/install.sh`
* Follow prompts for configuration.
* Check config with `bin/check_config.sh`
* Build with `bin/rebuild.sh`

## Other Commands

* `bin/start_dev_server.sh`
